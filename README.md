## 总述

JYRouter是一个基于[Routable](https://github.com/clayallsopp/routable-ios)封装的路由跳转库，封装了常见的Push,Pop,Present,Dismiss等跳转方法。

## 安装

### 手动安装

1. 下载 JYRouter 文件夹内的所有内容。
2. 将 JYRouter 内的源文件添加(拖放)到你的工程。
3. 导入 `JYRouter.h`.
